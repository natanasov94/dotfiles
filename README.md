## Installation
Clone the repo file

```sh
git clone https://github.com/natanasov94/dotfiles /tmp/dotfiles
```
Then source it
```sh
source /tmp/dotfiles/bootstrap.sh
```
And finally refresh the shell
```sh
exec bash
```

This will install all needed packages for the languages i am currently using
